1. Set majorita3.bdf (bdf=Block Diagram File) as Top-level entity
   In Project Navigator window, click bottom tab [Files] File
	Right mouse on majorita3.bdf -> Set as Top-Level Entity
2. Start Compilation - toolbar or  menu: Processing->Start Compilation
   or Crtl-L
3. Open majorita3_Waveform.vwf (vwf=Vector WaveForm)
4. Its menu: Simulation->Run functional Simulation
